---

## 🌟 What is BMS?

A complete, deployment-ready **web application** for managing microfinance 
and cooperative society branch operations — built from scratch using:

- **Google Apps Script** (302 server-side functions)
- **HTML5 + CSS3 + JavaScript** (3 separate portals)
- **Google Sheets** (36-sheet database)

---

## 👥 5 User Roles

| Role | Portal | Responsibility |
|------|--------|----------------|
| Admin | AdminLogin | Branch creation, user management, system-wide MIS |
| Supervisor | Main App | Approve entries, day operations, reports |
| Operator | Main App | Member registration, deposits, loans, repayments |
| Member | Main App | View accounts, statements, calculators |
| Treasurer | Cash Office | Cash transfers, branch daily reports |

---

## ✨ Key Features

- ✅ Complete member & account management with approval workflow
- ✅ SIP, Lumpsum, Fixed Deposit & Savings account handling
- ✅ Loan management — EMI, Flexible, NPA aging analysis
- ✅ 21 reports — Daily, Monthly, Audit, MIS, Operator-wise
- ✅ Cash Office module — branch ↔ office transfers
- ✅ Holiday auto-announcement with email to all members & staff
- ✅ Session-based security + salted SHA-256 password hashing
- ✅ Passbook printing, birthday wishes, OTP password reset
- ✅ Branch-level data isolation — multi-branch ready

---

## 🔒 Security Highlights

- Salted SHA-256 password hashing
- 8-hour session token validation
- Login lockout after failed attempts
- Backdate prevention (server-side)
- Every deletion audit-logged with reason
